## Instalation

You require Python 3.5 or greater.

Create your virtual environment. And install the dependencies.

```sh
(my_env)$ pip install -r requirements.txt
```

And run your server.

```sh
(my_env)$ python generador_memes/manage.py runserver
```

And enjoy.
